# Wireframe Proposal Spec C: Bookish Spread

## Use

This is a prompt-ready wireframe specification for an external tool such as Google AI Studio. It must produce an interactive wireframe, not polished final UI art.

The proposal must follow the governing spec in `docs/internal/ui-ux-spec-pamphlet-workbench.md`.

## Intent

This variant leans hardest into the pamphlet / open-book metaphor while preserving the same function and shape.

Its personality is:

- symmetrical
- composed
- page-like
- calm

It should feel unmistakably like an intentional two-page spread, not a generic split-pane app.

## Fixed shape

Keep the same product grammar:

- top strip
- upstream sliver
- two-page spread
- audit ledger sliver

The difference is not structure. The difference is how clearly the central spread reads as facing pages.

## Surface behavior

### Top strip

Keep it present but visually recessive.

It should feel like the instrument header above a reading spread, not the main attraction.

### Left upstream sliver

It should feel like a binding-adjacent margin.

Show:

- source context
- small metrics
- bottom-tucked file tree affordance

### Primary Surface

It should read like the left page of a book:

- selection-heavy
- review-heavy
- table-heavy

### Secondary Surface

It should read like the facing right page:

- consequence-heavy
- preview-heavy
- commit-heavy

### Audit ledger sliver

Treat it as the final downstream margin.

When expanded, the right page becomes a ledger reading surface.

## Workflows to include

Same four workflows:

- Normalize
- Delete Junk
- Repair Defaults
- Dashboard

## Visual treatment

Keep it stripped down but slightly more page-like than the other proposals:

- clearer central spread division
- more obvious page headers
- slightly calmer spacing rhythm
- stronger sense of left-page / right-page identity

Still remain a wireframe:

- no polished branding
- no final color system
- no decorative art

## What should vary in this proposal

Compared to the other three proposals, this one should:

- emphasize the symmetry of the spread most strongly
- make the two main surfaces feel most like intentional facing pages
- test whether the metaphor improves orientation without reducing density
